# Dia de IA · Edwards Lifesciences

Hub com os dois materiais do Dia de IA:

- `index.html` — página inicial (hub) com os links
- `manha/` — Workshop de Líderes (4 níveis + Copilot, 20 líderes, 4h)
- `tarde/` — Palestra do Time (IA First na prática, time administrativo, 1h)

## Publicar no GitHub Pages
1. Crie um repositório público e faça upload de **todo o conteúdo desta pasta** (mantendo as subpastas `manha/` e `tarde/`).
2. Settings → Pages → Source: Deploy from a branch → branch `main`, pasta `/ (root)` → Save.
3. O hub abre em `https://SEU-USUARIO.github.io/SEU-REPO/`, e os materiais em `/manha/` e `/tarde/`.

Estrutura:
```
/index.html
/manha/index.html
/tarde/index.html
```
